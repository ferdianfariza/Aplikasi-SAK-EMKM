Requirement:
<br>
PHP 8.2 Ke Atas
<br>
Composer 2.8.6
<br>
Laravel 12.0 Ke Atas


## Clone this project 
`git clone https://github.com/ferdianfariza/lelesegar.com.git`

## Open file directory and install dependencies
`composer install`
<br>
`php artisan migrate`
<br>
`php artisan db:seed`

## Run the app
`php artisan serve`
